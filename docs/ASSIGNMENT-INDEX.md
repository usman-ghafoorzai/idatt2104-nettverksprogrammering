# Assignment Index

This table maps the main assignment folders/reports to purpose and technologies.

| Path | Purpose | Main technology |
| --- | --- | --- |
| `datakommunikasjon/ovinger/DK-ovinger-1-2-3.pdf` | Report bundle for data communication assignments 1-3 | Written report (PDF) |

## Notes
- Assignments are independent exercises, not one integrated production project.
- The HTTPS exercise uses a local demo keystore file for coursework testing.
